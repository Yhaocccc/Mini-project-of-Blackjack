# Mini-project-of-Blackjack
這是由我親自操刀的 21 點小專題，透過 C++ 展現了物件導向設計的實務應用。 核心架構與技術遊戲主要由 Card 與 Deck 兩個類別組成：Card 類別：封裝單張撲克牌的資訊，巧妙地以 double 記錄牌值，整數部分代表點數（Rank），小數部分則區分花色（Suit）。Deck 類別：負責整副牌的管理，包含建立 52 張牌、洗牌（Shuffle）以及發牌邏輯。 靈活點數計算：程式模擬了A牌的特殊性，玩家可依戰略需求將 A 設為 1 點或 11 點。  莊家人工智慧：嚴格執行莊家點數小於 17 點必須強制加牌的標準規則。多樣化勝負判定：除了傳統的比大小，亦實作了「過五關」的特殊獲勝條件。這項專題不僅是小遊戲，更是將資料結構（如陣列、指標應用）與邏輯判斷結合的優秀實踐 。
